# 🌐 Eliganti Bala Krishna — Personal Portfolio

A fully responsive, animated personal portfolio website built with pure HTML, CSS, and JavaScript — featuring an **Admin Mode** for dynamic content management.

🔗 **Live Site:** [View Portfolio](https://balakrishna0457.github.io/MyPortfoilo)

---

## ✨ Features

- 🎨 **Animated UI** — particle background, floating profile ring, typewriter effect, scroll reveal animations
- 🔐 **Admin Mode** — password-protected panel to add/remove projects, skills, certifications, education, and publications dynamically
- 📁 **Dynamic Content Management** — manage all sections without touching code
- 📬 **Contact Form** — integrated with Formspree for direct email delivery
- 📱 **Fully Responsive** — works on all screen sizes
- 🖼️ **Profile Photo Upload** — upload your photo directly from the browser
- 📊 **Live Stats Counter** — auto-updates projects, certifications, and publication counts
